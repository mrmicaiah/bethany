# Bethany's Library

A simple dashboard to read Bethany's work.

## Setup

This is a Cloudflare Pages site that reads from Bethany's R2 bucket.

## Endpoints

- `/` - Home, current status
- `/books` - List all books
- `/books/:id` - Book detail with chapters
- `/books/:id/:chapter` - Read a chapter
- `/craft` - Her style guide and process
- `/ideas` - Her sparks and ideas
